## 🐛 常见问题排查

### 问题1：部门列表为空
**可能原因**：
- Token失效或未登录
- 后端服务未启动
- 数据库连接失败

**解决方案**：
```javascript
// 1. 检查Token
console.log('Token：', localStorage.getItem('token'))

// 2. 重新登录
localStorage.clear()
location.reload()
// 使用 admin/123456 登录

// 3. 检查后端服务
// 访问：http://localhost:8080/department/list
// 应该返回JSON数据或401错误
```

---

### 问题2：员工数量显示为0
**可能原因**：
- countEmployees接口调用失败
- 权限不足（部门主管查看其他部门）

**解决方案**：
```javascript
// 查看控制台警告日志
// 如果显示"获取部门X员工数量失败"，检查：
// 1. 是否有权限访问该部门
// 2. 后端是否有异常

// 手动测试接口
const token = localStorage.getItem('token')
const res = await fetch('http://localhost:8080/department/countEmployees/1', {
  headers: { 'Authorization': `Bearer ${token}` }
})

console.log('员工数量：', data)
```

---

### 问题3：部门主管只能看到自己部门
**这是正常行为**！

**权限设计**：
- 系统管理员：可以查看所有部门
- 部门主管：只能查看本部门
- 普通员工：只能查看自己所属部门

**如果需要让所有员工查看所有部门**，需要修改后端权限逻辑。
