# 前后端员工管理对接完成报告

## 📋 完成内容

### 1. 创建的文件

#### ✅ utils/request.js
- **功能**：axios 实例配置和拦截器
- **特性**：
  - 基础URL：`http://localhost:8080`
  - 请求拦截器：自动添加 JWT Token 到请求头
  - 响应拦截器：统一处理业务错误和HTTP错误
  - 401自动跳转登录页

#### ✅ api/employee.js
- **功能**：员工相关API接口封装
- **接口列表**：
  - `login(data)` - 员工登录
  - `getEmployeeList(page, size, departmentId)` - 获取员工列表（分页）
  - `getEmployeeById(id)` - 查询员工详情
  - `searchEmployee(key, value, page, size)` - 搜索员工
  - `getEmployeeByDepartment(departmentId)` - 按部门查询
  - `addEmployee(data)` - 添加员工
  - `updateEmployee(data)` - 更新员工信息
  - `deleteEmployee(id)` - 删除员工
  - `updatePassword(data)` - 修改密码
  - `updateEmployeeStatus(data)` - 更新员工状态
  - `updateEmployeeRole(data)` - 更新员工角色

## 🚀 使用说明

### 1. 启动后端
```bash
cd backend/oa-springboot
mvn spring-boot:run
```
后端启动在：http://localhost:8080

### 2. 启动前端
```bash
cd frontend/oa-mobile
npm install  # 首次运行需要安装依赖
npm run dev
```
前端启动在：http://localhost:5173
