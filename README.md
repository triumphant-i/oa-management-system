# OA办公管理系统

## 项目简介

这是一个完整的OA（办公自动化）管理系统，改造自原有的宿舍管理系统。系统采用前后端分离架构，提供了员工管理、部门管理、审批管理、公告管理、考勤管理、会议室管理等完整的办公自动化功能。

## 系统特点

- ✅ **前后端分离**: 前端Vue.js + 后端Spring Boot
- ✅ **权限管理**: 支持管理员和员工两种角色
- ✅ **功能完整**: 包含6大核心功能模块
- ✅ **界面美观**: 使用Element UI组件库，界面简洁大方
- ✅ **易于扩展**: 模块化设计，便于功能扩展
- ✅ **数据库完整**: 提供完整的数据库设计和初始化脚本

## 核心功能模块

### 1. 员工管理模块
- 员工信息的增删改查
- 员工登录认证
- 密码修改
- 按部门查询员工

### 2. 部门管理模块
- 部门信息的增删改查
- 部门员工数量统计
- 删除部门前的员工检查

### 3. 审批管理模块
- 提交各类申请（请假、出差、加班、报销、采购）
- 审批流程管理
- 申请历史查询
- 申请撤回

### 4. 公告管理模块
- 发布和管理公告
- 公告置顶功能
- 按分类查询公告
- 公告撤回

### 5. 考勤管理模块
- 签到/签退功能
- 考勤记录查询
- 迟到/早退判断
- 补卡申请

### 6. 会议室管理模块
- 会议室信息管理
- 会议室预约
- 时间冲突检测
- 预约记录查询

## 用户角色

### 管理员角色
- 员工管理：添加、编辑、删除员工信息
- 部门管理：管理部门信息
- 审批管理：审批员工提交的各类申请
- 公告管理：发布和管理公司公告
- 考勤管理：查看员工考勤记录
- 会议室管理：管理会议室资源

### 员工角色
- 个人信息：查看和修改个人信息
- 提交申请：提交请假、出差等申请
- 申请历史：查看自己的申请记录
- 公告列表：查看公司公告
- 考勤打卡：进行签到签退
- 会议室预约：预约会议室

## 快速开始

### 1. 环境准备
- JDK 17+
- Node.js 14+
- MySQL 8.0+
- Maven 3.6+
- npm 或 yarn

### 2. 数据库初始化
```bash
# 登录MySQL
mysql -u root -p

# 执行数据库脚本
source database/oa_system.sql
```

### 3. 启动后端
```bash
# 进入后端目录
cd backend/oa-springboot

# 修改数据库配置（如果需要）
# 编辑 src/main/resources/application.yml

# 编译并启动
mvn clean install
mvn spring-boot:run
```

后端启动成功后访问：http://localhost:8080

### 4. 启动前端
```bash
# 进入前端目录
cd frontend/oa-vue

# 安装依赖
npm install

# 启动开发服务器
npm run serve
```

前端启动成功后访问：http://localhost:8080

## 数据库设计

数据库包含8张核心表：
1. `t_admin` - 管理员表
2. `t_employee` - 员工表
3. `t_department` - 部门表
4. `t_approval` - 审批表
5. `t_announcement` - 公告表
6. `t_attendance` - 考勤表
7. `t_meeting_room` - 会议室表
8. `t_meeting` - 会议预约表

详细的数据库设计请参考：[database/oa_system.sql](database/oa_system.sql)
